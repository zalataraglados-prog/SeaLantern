# SeaLantern 项目结构

## 根目录
- .github/ - GitHub 配置文件
- src/ - 源代码目录
- src-tauri/ - Tauri 相关代码
- .editorconfig - 编辑器配置
- .gitattributes - Git 属性配置
- .gitignore - Git 忽略文件
- .oxfmtrc.json - 格式化配置
- .oxlintrc.json -  lint 配置
- AI_GUIDE.md - AI 指南
- CONTRIBUTING-en.md - 英文贡献指南
- CONTRIBUTING.md - 中文贡献指南
- Cargo.lock - Rust 依赖锁文件
- Cargo.toml - Rust 项目配置
- LICENSE - 许可证文件
- README-en.md - 英文 README
- README.md - 中文 README
- i18n.md - 国际化文档
- index.html - 项目入口 HTML
- package-lock.json - npm 依赖锁文件
- package.json - npm 项目配置
- rustfmt.toml - Rust 格式化配置
- tsconfig.json - TypeScript 配置
- tsconfig.node.json - TypeScript Node 配置
- vite.config.ts - Vite 配置
- 新手使用教程.html - 新手使用教程
- 项目结构.txt - 项目结构文档

## src 目录
- api/ - API 接口定义
  - config.ts - 配置相关 API
  - java.ts - Java 相关 API
  - player.ts - 玩家相关 API
  - server.ts - 服务器相关 API
  - settings.ts - 设置相关 API
  - system.ts - 系统相关 API
  - tauri.ts - Tauri 调用封装
  - update.ts - 更新相关 API
- assets/ - 静态资源
  - logo.svg - 项目 logo
  - vue.svg - Vue logo
- components/ - 组件目录
  - common/ - 通用组件
    - SLBadge.vue - 徽章组件
    - SLButton.vue - 按钮组件
    - SLCard.vue - 卡片组件
    - SLInput.vue - 输入框组件
    - SLModal.vue - 模态框组件
    - SLNotification.vue - 通知组件
    - SLProgress.vue - 进度条组件
    - SLSelect.vue - 选择器组件
    - SLSpinner.vue - 加载组件
    - SLSwitch.vue - 开关组件
    - UpdateModal.vue - 更新模态框
  - config/ - 配置相关组件
    - ConfigToolbar.vue - 配置工具栏
    - ConfigCategories.vue - 配置分类选择
    - ConfigEntry.vue - 配置项组件
  - console/ - 控制台相关组件
    - ConsoleToolbar.vue - 控制台工具栏
    - ConsoleCommands.vue - 控制台指令组件
    - ConsoleOutput.vue - 控制台输出组件
    - ConsoleInput.vue - 控制台输入组件
    - CommandModal.vue - 命令编辑模态框
  - layout/ - 布局组件
    - AppHeader.vue - 应用头部
    - AppLayout.vue - 应用布局
    - AppSidebar.vue - 应用侧边栏
  - splash/ - 启动屏幕组件
    - SplashScreen.vue - 启动屏幕
- data/ - 数据文件
  - contributors.ts - 贡献者数据
- locales/ - 国际化文件
  - en-US.json - 英文语言包
  - index.ts - 国际化配置
  - zh-CN.json - 中文语言包
  - zh-JB.json - 中文（简化）语言包
  - zh-TW.json - 繁体中文语言包
- router/ - 路由配置
  - index.ts - 路由定义
- stores/ - Pinia 状态管理
  - consoleStore.ts - 控制台状态
  - i18nStore.ts - 国际化状态
  - index.ts - 状态管理配置
  - serverStore.ts - 服务器状态
  - settingsStore.ts - 设置状态
  - uiStore.ts - UI 状态
  - updateStore.ts - 更新状态
- styles/ - 样式文件
  - animations.css - 动画样式
  - glass.css - 玻璃效果样式
  - reset.css - 重置样式
  - typography.css - 排版样式
  - variables.css - 变量样式
- types/ - TypeScript 类型定义
  - common.ts - 通用类型
  - server.ts - 服务器相关类型
- utils/ - 工具函数
  - constants.ts - 常量定义
  - errorHandler.ts - 错误处理
  - logger.ts - 日志工具
  - serverStatus.ts - 服务器状态工具
  - tray.ts - 托盘功能
  - version.ts - 版本工具
- views/ - 页面组件
  - AboutView.vue - 关于页面
  - ConfigView.vue - 配置页面
  - ConsoleView.vue - 控制台页面
  - CreateServerView.vue - 创建服务器页面
  - HomeView.vue - 首页
  - PaintView.vue - 个性化页面
  - PlayerView.vue - 玩家管理页面
  - SettingsView.vue - 设置页面
- App.vue - 应用根组件
- main.ts - 应用入口
- style.css - 全局样式
- vite-env.d.ts - Vite 环境类型定义

## src-tauri 目录
- capabilities/ - Tauri 能力配置
  - default.json - 默认能力配置
- icons/ - 应用图标
  - android/ - Android 图标
  - ios/ - iOS 图标
  - 其他平台图标文件
- src/ - Rust 源代码
  - commands/ - Tauri 命令
    - config.rs - 配置相关命令
    - java.rs - Java 相关命令
    - join.rs - 加入相关命令
    - mod.rs - 命令模块
    - mods.rs - 模组相关命令
    - player.rs - 玩家相关命令
    - server.rs - 服务器相关命令
    - server_id.rs - 服务器 ID 相关命令
    - settings.rs - 设置相关命令
    - system.rs - 系统相关命令
    - update.rs - 更新相关命令
  - models/ - 数据模型
    - config.rs - 配置模型
    - mod.rs - 模型模块
    - server.rs - 服务器模型
    - settings.rs - 设置模型
  - services/ - 服务
    - config_parser.rs - 配置解析服务
    - global.rs - 全局服务
    - java_detector.rs - Java 检测服务
    - join_manager.rs - 加入管理器
    - mod.rs - 服务模块
    - mod_manager.rs - 模组管理器
    - player_manager.rs - 玩家管理器
    - server_id_manager.rs - 服务器 ID 管理器
    - server_manager.rs - 服务器管理器
    - settings_manager.rs - 设置管理器
  - utils/ - 工具函数
    - cli.rs - CLI 工具
    - downloader.rs - 下载工具
    - mod.rs - 工具模块
  - lib.rs - 库入口
  - main.rs - 应用入口
- .gitignore - Git 忽略文件
- Cargo.lock - Rust 依赖锁文件
- Cargo.toml - Rust 项目配置
- build.rs - 构建脚本
- tauri.conf.json - Tauri 配置文件
